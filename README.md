# 🎬 Movie Maniac (React)

A modern and responsive Movie App built using React.js that displays a list of movies with real-time data.

---

## 🚀 Features

* 🎥 Display a list of movies
* ⭐ View movie details (title, rating, overview, poster)
* 📱 Fully responsive design
* ⚡ Fast performance using React
* 🌐 API integration for real-time movie data
* 🎨 Clean and user-friendly UI

---

## 🛠️ Tech Stack

* **Frontend:** React.js, JavaScript, HTML, CSS
* **API:** TMDB (The Movie Database) API
* **Styling:** CSS / Tailwind (if used)

## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/your-username/movie-app.git
```

2. Navigate to project folder:

```bash
cd movie-app
```

3. Install dependencies:

```bash
npm install
```

4. Start development server:

```bash
npm run dev
```
